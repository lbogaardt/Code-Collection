# Simulation Studies of Statistical Models  

This repository contains multiple **simulation studies** implemented in `.Rmd` files. The `.Rmd` files are rendered to `.html` for easy viewing.  

## Repository Structure  
- **`simulations/`** – Contains `.Rmd` files for various simulation studies.  
- **`results/`** – Rendered `.html` outputs for each study.  
- **`data/`** – (Optional) Datasets used in simulations.  
- **`scripts/`** – Supporting R scripts, if applicable.  

## Simulation Topics  
The studies cover:  
- Regression models (linear, logistic, etc.)  
- Bayesian inference  
- Monte Carlo methods  
- Machine learning approaches  

## Viewing Results  
Rendered `.html` files can be found in the **[`results/`](results/)** folder or viewed directly via **GitHub Pages** (if enabled).
